## TCG Companion
* How to build and run the website
  - First, change the api key in appsettings.json to the api key I sent you, where it says Gemini ApiKey: **"YOUR KEY HERE"**.
  - Second, when pulling the website, do **dotnet clean**
  - Third, build the website, with **dotnet build**
  - Last run the app with **dotnet run**
* Features
  - Deck build which holds the Gemini assistant that helps the user build a deck
  - Collection that stores the user collection and the cost of each card
  - Global chat that allows other users to talk with each other
  - A bio section that allows users to make their own display name for the global chat
